# JavaScript Fundamentals – Week 1

> A collection of foundational JavaScript programs focusing on core programming concepts, problem-solving, and algorithmic thinking.

## Overview

This folder contains a curated set of beginner-level JavaScript exercises. These programs were implemented to strengthen programming fundamentals through hands-on practice with:
- **Control Flow:** Conditional statements and logic.
- **Data Structures:** Array manipulation and object operations.
- **Modularity:** Function implementation and scope.
- **Problem Solving:** Mathematical logic and search algorithms.

## Technologies & Tools

- **Language:** JavaScript (ES6+)
- **Runtime:** Node.js
- **Editor:** Visual Studio Code

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Execution Instructions

1. Open your terminal or command prompt.
2. Navigate to the directory containing the files.
3. Run any file using the following command:

```bash
node "filename.js"
```

**Example:**
```bash
node "biggestOfTwoNumbers Ques1.js"
```
