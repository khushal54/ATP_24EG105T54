# Product Listing Page 

A responsive product showcase application built with React and Vite. This project demonstrates how to render a list of products dynamically using mapping and modular components.

## Features
- **Dynamic Product Grid**: Displays a collection of products using a responsive CSS grid.
- **Product Cards**: Each card includes an image, title, price, description, and rating.
- **Modular Architecture**: Uses a dedicated `Product` component for clean and maintainable code.
- **Tailwind CSS Styling**: Utilizes Tailwind CSS for modern, utility-first styling.

## Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS 4

## Project Structure
- `src/App.jsx`: Main entry point containing the product data array and grid layout.
- `src/components/Product.jsx`: Functional component for rendering individual product cards.
- `src/index.css`: Global styles including Tailwind directives.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
