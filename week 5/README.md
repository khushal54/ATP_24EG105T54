# Week 5 – CSS Layouts and React Basics

## Overview
This folder contains exercises and practice programs focused on webpage layouts using CSS and introductory concepts of React. The work spans three days covering HTML table construction, Flexbox-based layouts, responsive design with media queries, and building component-driven React applications with Vite and Tailwind CSS.

## Folder Structure

###  16-03-2026

#### Ques 1 – HTML Tables 
- **Ques 1.1 – Title Table**  — Basic HTML table demonstrating title/header structuring with CSS styling.
- **Ques 1.2 – Subjects Table**  — Tabular representation of subject data with styled rows and columns.
- **Ques 1.3 – Users Table**  — Styled user information table with advanced CSS formatting.

#### Ques 2 – Flexbox Layout 
- A complete webpage layout built entirely using CSS Flexbox for flexible content arrangement and alignment.


### 17-03-2026

#### Ques 1 – Normal Layout 
- A standard webpage layout built with conventional CSS positioning and styling techniques.

#### Ques 2 – Responsive Layouts 
- **Ques 2.1 – Responsive Layout 1**   — Responsive webpage using media queries for adaptive design across screen sizes.
- **Ques 2.2 – Responsive Layout 2**  — A more complex responsive layout with extended media query breakpoints.


### 18-03-2026

#### React App 1 – Product Listing Grid 
- A product catalog application rendering a grid of product cards using data from the Fake Store API structure. Demonstrates **props**, **`map()` rendering**, and the **reusable `Product` component**.

#### React App 2 – Users List Application 
- A multi-component app with **Navbar**, **UsersList**, **User**, and **Footer** components. Demonstrates component composition, data fetching, and structured page layout in React.

#### React App 3 – Profile Cards 
- A profile display application with **Header** and **Profiles/Profile** components. Demonstrates component hierarchy, reusable card patterns, and clean layout composition.


## Topics Covered

- HTML tables and structured data display
- CSS Flexbox for flexible layouts
- Responsive web design with media queries
- Standard CSS positioning and styling
- React fundamentals (JSX, components, props)
- Component composition and reusability
- List rendering with `map()`
- Vite-based React project setup

## How to Run
